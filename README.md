# 强化学习的数学原理学习笔记

这是《强化学习的数学原理》的个人学习笔记仓库，内容围绕全书 10 章展开，包含章节切片、逐章讲解、补充推导和二轮复习总结。

## 快速入口

- [全书地图与问题链](_tutor/二轮总结/00_全书地图与问题链.md)：从全局问题链理解 10 章之间的关系。
- [概念对比表](_tutor/二轮总结/01_概念对比表.md)：集中对比容易混淆的概念、算法和符号。
- [证明推导索引](_tutor/二轮总结/02_证明推导索引.md)：按证明、引理和关键推导查找细节。
- [易错点与统一符号表](_tutor/二轮总结/03_易错点与统一符号表.md)：复习符号体系和常见误区。

## 推荐读法

1. 先读 [00_全书地图与问题链](_tutor/二轮总结/00_全书地图与问题链.md)，建立“从 MDP 到 Actor-Critic”的主线。
2. 复习某一章时，先打开 `_tutor/二轮总结/章节地图/chXX.html` 看该章内部结构。
3. 卡在概念或推导时，再进入 `_tutor/notes/` 中对应章节笔记和 supplement 文件。


## 章节主线

全书可以理解为在不断放松“模型已知、状态有限、完整回报可得、策略可枚举”等假设：

| 章节 | 主题 |
|---|---|
| Ch1 | 基本概念与 MDP |
| Ch2 | 状态值与贝尔曼方程 |
| Ch3 | 最优策略与贝尔曼最优方程 |
| Ch4 | 值迭代、策略迭代与动态规划 |
| Ch5 | 蒙特卡洛方法 |
| Ch6 | 随机近似与 SGD |
| Ch7 | 时序差分方法 |
| Ch8 | 值函数近似与 DQN |
| Ch9 | 策略梯度 |
| Ch10 | Actor-Critic 方法 |
