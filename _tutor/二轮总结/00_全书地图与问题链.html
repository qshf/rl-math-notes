1-10 章主问题：怎样从“智能体和环境交互”这件事，一步步走到不用模型、能在线学习、能处理大规模状态，并能直接优化策略？
一句话答案：Ch1 把问题翻译成 MDP；Ch2-Ch3 写出策略评估与最优性的 Bellman 方程；Ch4-Ch7 从模型已知走到 model-free TD；Ch8 把表格价值换成函数近似；Ch9 直接参数化策略；Ch10 让 actor 的策略梯度和 critic 的 TD 价值学习合体。
Ch1 语言MDP：状态、动作、奖励、策略、转移、回报 Ch2 评估给定 $\pi$，用 $v_\pi,q_\pi$ 和 BE 衡量好坏 Ch3 最优把按策略平均换成取最大，得到 BOE Ch4 精确解模型已知：VI / PI / GPI 解 BOE Ch5 采样解模型未知：用 return 平均估 $q_\pi$ Ch6 地基样本平均升级成随机近似和 SGD Ch7 在线解TD：一步样本 + 自举 target + 增量更新 Ch8 泛化用 $\hat v,\hat q$ 和参数 $w$ 代替表格 Ch9 策略直接学 $\pi(a|s,\theta)$，用 PG / REINFORCE Ch10 合体Actor-Critic：actor 改策略，critic 给信号

主线问题链

Ch1 · MDP 先把“学习”翻译成可计算的随机过程
本章回答
智能体处在状态 $s$，选择动作 $a$，环境按概率 $p(s',r|s,a)$ 给出下一状态和奖励，策略 $\pi(a|s)$ 决定动作分布。长期目标不是单步奖励，而是折扣回报 $G_t$。
接力棒 有了 MDP 语言，下一步才能问：给定一个策略 $\pi$，它到底好不好？
▼给定策略，怎么定量评估？
Ch2 · BE 用价值函数评估一个策略
本章回答
$v_\pi(s)$ 是从状态 $s$ 出发、按策略 $\pi$ 行动的期望回报；$q_\pi(s,a)$ 是先做动作 $a$ 后再按 $\pi$ 行动的期望回报。Bellman equation 把“长期回报”拆成“一步奖励 + 下个状态的价值”。
接力棒 Ch2 只会评估一个给定策略；下一步要问：所有策略里最好的那个怎么刻画？
▼从评估一个策略，升级到寻找最优策略
Ch3 · BOE 把“按策略平均”换成“取最大”
本章回答
最优值 $v_*,q_*$ 满足 Bellman optimality equation。和 Ch2 的 BE 相比，关键变化是：不再按固定 $\pi$ 对动作加权，而是在每个状态里选能让未来价值最大的动作。
接力棒 BOE 刻画了最优解，但它非线性，不能像线性 BE 那样直接求逆；下一步要把它变成算法。
▼BOE 怎么实际求？先假设模型已知
Ch4 · DP / GPI 模型已知时，用迭代解 Bellman 方程
本章回答
值迭代直接反复套 Bellman optimality operator；策略迭代交替做 policy evaluation 和 policy improvement；截断策略迭代把两者连成一条连续谱。统一名字叫 GPI：评估一点，改进一点，再评估。
接力棒 Ch4 的硬前提是已知 $p,r$。现实里模型常常未知，下一步只能靠采样。
▼模型未知，期望算不出怎么办？
Ch5 · MC 用完整后续回报的平均替代模型期望
本章回答
$q_\pi(s,a)$ 本质是回报的期望。模型未知时，不计算转移概率，只收集很多条完整 episode，把访问到的 $(s,a)$ 后续回报拿来平均。MC Exploring Starts、every-visit、$\epsilon$-greedy 都是在解决覆盖与样本效率。
接力棒 MC 摆脱了模型；every-visit / 逐 episode 版本可以一条 episode 结束后立刻更新沿途所有 visit，但仍要先知道这些 visit 的完整后续回报 $G_t$。下一步要问：能不能不等完整回报，只来一步就更新？
▼样本一个个来，怎样增量更新还保证收敛？
Ch6 · SA 随机近似给所有“旧值 + 步长 × 误差”打地基
本章回答
样本平均可以写成 $w_{k+1}=w_k+\alpha_k(x_k-w_k)$。Robbins-Monro 把它推广为带噪求根；Dvoretzky 给误差收敛模板；SGD 是把随机近似用于优化。它不是又一个 RL 算法，而是后面 TD、函数近似、策略梯度的共同数学语法。
接力棒 现在我们知道增量更新为什么合理；下一步把 Bellman target 也做成一步样本 target。
▼能不能不等完整后续回报，一步样本到手就学？
Ch7 · TD 采样 + 自举 + 随机近似，合成在线 model-free 学习
本章回答
TD target 用“一步真实奖励 + 当前价值估计”替代完整回报。Sarsa 用实际下一动作学习 $q_\pi$；Expected Sarsa 把下一动作先求期望；$n$-step Sarsa 调节真实展开步数；Q-learning 用 $\max_{a'}q(s',a')$ 直接朝 $q_*$ 学。
接力棒 到 Ch7，表格型 RL 的主线闭合了：不用模型、在线更新、可做策略控制。下一大问题是：状态/动作太多，表格存不下怎么办？
▼状态太多，表格存不下怎么办？
Ch8 · Value Function Approximation 把表格价值换成参数化函数
本章回答
不再给每个 $s$ 或 $(s,a)$ 存一个格子，而是用 $\hat v(s,w)$、$\hat q(s,a,w)$ 表示价值。更新从“改一个格子”变成“用 MC return 或 TD target 最小化预测误差，更新参数 $w$”。线性 TD、LSTD 和 DQN 都是这条线的不同落点。
接力棒 函数近似解决了价值表太大的问题，但仍常常是“先学价值，再从价值导出策略”。下一步问：能不能直接学习策略本身？
▼能否跳过学值，直接参数化策略？
Ch9 · Policy Gradient 把策略写成 $\pi(a|s,\theta)$，沿表现指标的梯度上升
本章回答
先选标量指标 $J(\theta)$ 衡量策略好坏，再用 policy gradient theorem 把 $\nabla_\theta J$ 写成可采样的期望 $\mathbb E[\nabla_\theta\ln\pi(A|S,\theta)q_\pi(S,A)]$。REINFORCE 用完整回报 $G_t$ 估计动作价值，得到最朴素的 Monte Carlo 策略梯度。
接力棒 策略梯度能直接改策略，但 $q_\pi$ 用 MC return 估时方差高、更新晚。下一步要把 value learner 接回来，专门给 actor 提供评价信号。
▼值方法与策略方法能否合体？
Ch10 · Actor-Critic actor 负责改策略，critic 负责在线估评价信号
本章回答
QAC 把 REINFORCE 的 MC return 换成 TD critic 的 $q(s,a,w)$；A2C 用 baseline / advantage 把绝对价值换成相对好坏；off-policy AC 用 importance weight 处理样本来自 $\beta$；deterministic AC 让 actor 直接输出连续动作并沿 $\nabla_a q$ 改动作。
阶段终点 到 Ch10，value-based 与 policy-based 两条线合流：critic 提供低方差评价，actor 提供可微策略。这就是许多现代深度 RL 算法的骨架。

Ch1
MDP 语言		$M=(\mathcal S,\mathcal A,p,r,\gamma)$
$G_t=\sum_{k=0}^{\infty}\gamma^kR_{t+k+1}$		RL 不是只看当前奖励，而是在 MDP 里最大化长期折扣回报。既然目标是长期回报，给定策略 $\pi$ 时如何算它的期望回报？
Ch2
Bellman equation		$v_\pi(s)=\mathbb E_\pi[R_{t+1}+\gamma v_\pi(S_{t+1})\mid S_t=s]$长期回报可递归拆成“一步奖励 + 下个状态价值”，所以策略评估变成解自洽方程。如果不只是评估一个 $\pi$，而是要找最好的 $\pi$，方程怎么变？
Ch3
Bellman optimality equation		$q_*(s,a)=\mathbb E[R_{t+1}+\gamma\max_{a'}q_*(S_{t+1},a')\mid S_t=s,A_t=a]$把“按策略平均”换成“对动作取最大”，就从 $q_\pi$ 升级到最优动作值 $q_*$。BOE 给了最优性的方程，但这个非线性方程如何实际求解？
Ch4
DP / GPI		$q_\pi(s,a)=\sum_{s',r}p(s',r|s,a)[r+\gamma v_\pi(s')]$
$\pi_{new}(s)=\arg\max_a q_\pi(s,a)$		GPI 的两步在公式里很清楚：先用已知模型 $p,r$ 算出每个动作的期望回报 $q_\pi$，再对 $q_\pi$ 贪心改进策略。如果 $p(s',r|s,a)$ 不知道，第一步的模型期望算不了，还能不能得到 $q_\pi(s,a)$？
Ch5
Monte Carlo		$q_\pi(s,a)=\mathbb E_\pi[G_t\mid S_t=s,A_t=a]$
$q(s,a)\leftarrow \frac1{N(s,a)}\sum_{i=1}^{N(s,a)}G_i(s,a)$		Ch5 没换目标，仍是在给 GPI 估 $q_\pi$；只是把 Ch4 的模型期望 $\sum p(\cdot)[\cdot]$ 换成采样平均。every-visit 会在本条 episode 结束后反向算出所有 $G_t$ 并立刻更新。现在 $q_\pi$ 能免模型估了，但 $G_t$ 仍要等完整后续回报；能不能只用一步奖励加估计值来更新？
Ch6
随机近似		$w_{k+1}=w_k+\alpha_k(\text{sample}_k-w_k)$
$w_{k+1}=w_k-a_k\tilde g(w_k,\eta_k)$		样本平均、RM、SGD 都是“旧估计 + 步长 × 带噪误差”；步长条件控制噪声并保证收敛。如果 sample target 换成 Bellman target，能不能每一步环境交互后立刻更新价值？
Ch7
TD / Sarsa / Q-learning		$q_{t+1}(s_t,a_t)=q_t(s_t,a_t)+\alpha_t[\bar q_t-q_t(s_t,a_t)]$
$\bar q_t=r_{t+1}+\gamma\,[\text{下一步价值项}]$
算法下一步价值项
Sarsa$q_t(s_{t+1},a_{t+1})$
Expected Sarsa$\sum_{a'}\pi(a'|s_{t+1})q_t(s_{t+1},a')$
Q-learning$\max_{a'}q_t(s_{t+1},a')$
TD 把 Ch5 的采样和 Ch6 的增量更新合起来；统一壳子不变，不同算法只是在换 $\bar q_t$ 里的下一步价值项。到这里仍要存表格 $q(s,a)$；状态/动作太多时，下一章要用函数近似。
Ch8
值函数近似		$\hat y(x,w)\in\{\hat v(s,w),\hat q(s,a,w)\}$
$w_{t+1}=w_t+\alpha[\text{target}-\hat y(x_t,w_t)]\nabla_w\hat y(x_t,w_t)$
选择含义
$x_t=s_t$近似状态值 $\hat v(s,w)$
$x_t=(s_t,a_t)$近似动作值 $\hat q(s,a,w)$
target可用 MC return、TD target 或 DQN target
把表格值换成参数函数；更新不再只改一个格子，而是沿梯度改共享参数，让相似状态可以泛化。价值函数能近似了，但控制仍常是从价值导出策略；能不能直接优化策略函数？
Ch9
策略梯度		$\nabla_\theta J(\theta)=\mathbb E[\nabla_\theta\ln\pi(A|S,\theta)\,[\text{价值信号}]]$
$\theta\leftarrow\theta+\alpha[\text{价值信号}]_t\nabla_\theta\ln\pi(a_t|s_t,\theta)$
选择价值信号
理论 PG$q_\pi(S,A)$
REINFORCE$G_t$ 近似 $q_\pi(s_t,a_t)$
带 baseline$G_t-b(s_t)$ 或 $q_\pi-v_\pi$
把策略本身参数化，用 log-trick 把目标梯度写成可采样期望；本章最朴素实现用 MC return 估 $q_\pi$。MC 估 $q_\pi$ 方差大；能不能用 TD critic 在线给策略梯度提供低方差信号？
Ch10
Actor-Critic		$\theta_{t+1}=\theta_t+\alpha_\theta[\text{actor 信号}]_t$
$w_{t+1}=w_t+\alpha_w[\text{critic 信号}]_t$
算法actor 信号 / critic 信号
QAC$\nabla_\theta\ln\pi\,q(s_t,a_t,w_t)$ / $\delta_t\nabla_wq$
A2C$\delta_t\nabla_\theta\ln\pi$ / $\delta_t\nabla_wv$
Off-policy AC$c_t\delta_t\nabla_\theta\ln\pi$ / $c_t\delta_t\nabla_wv$
Deterministic AC$\nabla_\theta\mu\,\nabla_aq$ / $\delta_t\nabla_wq$
actor 用策略梯度改策略，critic 用 TD 学价值或 advantage；A2C、off-policy AC、deterministic AC 都是在换 critic 信号或策略形式。本书主线合流：现代 RL 多是在 AC 骨架上继续加稳定化、约束和探索机制。
关键桥 Ch4 → Ch7 是“同一个 Bellman / GPI 目标，逐步换估计方式”；Ch8 → Ch10 是“同一个增量更新壳子，逐步把表格值换成函数、把贪心策略换成可微策略、再把两条线合成 actor-critic”。

四条公式主线

1. 策略评估：给定 $\pi$，解 Bellman equation
$$ v_\pi(s)=\mathbb E_\pi[R_{t+1}+\gamma v_\pi(S_{t+1})\mid S_t=s] $$

读法：当前状态的价值 = 一步奖励 + 下个状态价值。Ch2 的全部策略评估都围绕这句话。

2. 最优控制把按策略走，换成选最好动作
$$ q_*(s,a)=\mathbb E[R_{t+1}+\gamma\max_{a'}q_*(S_{t+1},a')\mid S_t=s,A_t=a] $$

Ch3 写出 BOE，Ch4 在模型已知时解它，Ch7 的 Q-learning 用样本近似它。

3. 增量学习所有在线算法的共同壳子
$$ \text{new estimate} = \text{old estimate} +\alpha(\text{sample target}-\text{old estimate}) $$

Ch6 证明这种带噪更新为什么能收敛；Ch7 的 TD、Sarsa、Q-learning 只是换不同的 sample target。

4. 参数化优化从价值参数 $w$ 到策略参数 $\theta$
$$ w\leftarrow w+\alpha\delta\nabla_w\hat v(s,w), \qquad \theta\leftarrow\theta+\alpha\,\widehat A\,\nabla_\theta\ln\pi(a|s,\theta) $$

Ch8 更新价值函数参数 $w$；Ch9 更新策略参数 $\theta$；Ch10 用 critic 的 $\delta$ 或 advantage 作为 actor 的更新信号。

MC、随机近似、TD 的关系不要混

方法target 从哪里来什么时候更新是否自举它在主线里的角色
MC完整后续回报 $G_t$本条 episode 结束后反向算 $G_t$，可立刻更新沿途所有 visit否第一次真正摆脱模型，用采样平均代替期望
随机近似任意带噪观测样本一来就能更新不特指给 $w\leftarrow w+\alpha(\cdot)$ 这类更新提供数学框架
TD$r+\gamma v(s')$ 或 $r+\gamma q(s',a')$每一步都能更新是，用当前估计接尾巴把 MC 的 model-free 和 Ch6 的增量更新合并起来
核心洞见
Ch1-Ch10 不是十个散章，而是一条逐步现实化的链：先定义问题，再定义价值和最优；再从模型已知的 DP 放松到 model-free 的 MC/TD；再从表格放松到函数近似；最后从“学值再导出策略”推进到“直接学策略”，并在 Actor-Critic 中让 TD critic 和 policy-gradient actor 合流。

二刷易错提醒
不要把“取最大”一概理解成“策略一定纯 greedy”。在 Q-learning 里，TD target 可以用 $\max_{a'}q(s',a')$；但实现时行为策略可以是 $\epsilon$-greedy、uniform，或其他探索策略。on-policy / off-policy 说的是行为策略和目标策略的关系；online / offline 说的是什么时候处理样本。 到 Ch10 还要再加一层：actor-critic 不是“不要价值函数”，而是把价值估计变成 critic，专门服务于 actor 的策略梯度。
